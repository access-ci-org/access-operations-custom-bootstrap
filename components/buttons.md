# Buttons

## ACCESS Core Design System Color Variants

**Normal Button**: The text color is changed to white or black for each variant based on the color contrast ratio on the
background color to maintain the accessibility standards.

**Outline Button**: The background color is changed to white or black for each variant based on the color contrast ratio
on the text color to maintain the accessibility standards.

| Variant Class Name     | Normal Button Variant                                                                 | Outline Button Variant                                                                                | Link Button Variant                                                                                |
|:-----------------------|:--------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| **`dark`**             | <button type="button" class="btn btn-dark">.btn-dark</button>                         | <button type="button" class="btn btn-outline-dark">.btn-outline-dark</button>                         | <button type="button" class="btn btn-link">Link</button>                                           |
| **`primary`**          | <button type="button" class="btn btn-primary">.btn-primary</button>                   | <button type="button" class="btn btn-outline-primary">.btn-outline-primary</button>                   | <button type="button" class="btn btn-link text-primary">Link</button>                              |
| **`primary-light`**    | <button type="button" class="btn btn-primary-light">.btn-primary-light</button>       | <button type="button" class="btn btn-outline-primary-light">.btn-outline-primary-light</button>       | <button type="button" class="btn btn-link text-primary-light">Link</button>                        |
| **`light`**            | <button type="button" class="btn btn-light">.btn-light</button>                       | <button type="button" class="btn btn-outline-light">.btn-outline-light</button>                       | <button type="button" class="btn btn-link text-light bg-dark px-2 rounded">Link</button>           |
| **`secondary-light`**  | <button type="button" class="btn btn-secondary-light">.btn-secondary-light</button>   | <button type="button" class="btn btn-outline-secondary-light">.btn-outline-secondary-light</button>   | <button type="button" class="btn btn-link text-secondary-light bg-dark px-2 rounded">Link</button> |
| **`secondary`**        | <button type="button" class="btn btn-secondary">.btn-secondary</button>               | <button type="button" class="btn btn-outline-secondary">.btn-outline-secondary</button>               | <button type="button" class="btn btn-link text-secondary">Link</button>                            |
| **`secondary-dark`**   | <button type="button" class="btn btn-secondary-dark">.btn-secondary-dark</button>     | <button type="button" class="btn btn-outline-secondary-dark">.btn-outline-secondary-dark</button>     | <button type="button" class="btn btn-link text-secondary-dark">Link</button>                       |
| **`accent-primary`**   | <button type="button" class="btn btn-accent-primary">.btn-accent-primary</button>     | <button type="button" class="btn btn-outline-accent-primary">.btn-outline-accent-primary</button>     | <button type="button" class="btn btn-link text-accent-primary">Link</button>                       |
